# **App Name**: JustlyAI

## Core Features:

- Chat Interface: Intuitive chat interface modeled after ChatGPT.
- Collapsible Sidebar: Collapsible sidebar to display and navigate between recent chats.
- Sticky Input Bar: Bottom-aligned sticky input bar for seamless message input across devices.
- Speech-to-Text: Microphone integration for real-time speech transcription.
- Login/Signup Authentication: Secure authentication flow with polished login and signup pages.

## Style Guidelines:

- Color scheme: Dark background (#1E1E1E) for a modern and immersive feel.
- Primary color: A vibrant purple (#A050BE) to convey innovation and creativity.
- Accent color: Analogous blue (#5082BE) to complement the purple and create visual interest.
- Font pairing: 'Space Grotesk' for headers combined with 'Inter' for body text.
- Modern and minimalist icons for common actions (send, microphone, settings, etc.)
- Clean and structured layout with clear separation of chat messages.
- Smooth transitions and animations for sidebar, chat loading, and message display.
